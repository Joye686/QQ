## 🚀 快速开始

1. **访问在线版本**：点击上方表格中的"打开"链接
2. **本地运行**：下载 HTML 文件，用浏览器直接打开
3. **数据对接**：修改 JS 中的 `mockData` 或 `loadData()` 函数接入真实数据

## 📊 V3.3 功能特性

### 核心功能
- ✅ **T 字型图表**：嘉兴仓 vs 海外仓对比分析
- ✅ **三级产品筛选**：一级 → 二级 → 三级联动选择
- ✅ **多维度分析表**：客户选择 vs 实际发出详细数据
- ✅ **国家/仓库筛选**：按国家、仓库、时间粒度筛选
- ✅ **8 大指标卡片**：单价/KG、单价/包裹、发货包裹数等

### 图表类型
- 📈 折线图（时间趋势）
- 🍩 环形图（占比分析）
- 📊 柱状图（对比分析）
- 📋 详细数据表
